Radius 0 es la regla en este sistema. Todos los componentes del UI kit (badges, tablas, filas, filtros, cards) usan border-radius: 0. El 2px y el 4px son el techo permitido por DESIGN.md §6 pero no se usan en producción. El zoom de la esquina muestra la curva real (×12) para que se note la diferencia entre los tres valores.
